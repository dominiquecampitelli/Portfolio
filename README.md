# Dominique Campitelli - Portifólio Windows 98

## Um portfólio interativo desenvolvido em React + TypeScript, inspirado no visual clássico do Windows 98, trazendo janelas, inventário e uma experiência nostálgica diretamente no navegador. O objetivo é unir design nostálgico, funcionalidade moderna e uma apresentação criativa das minhas habilidades como desenvolvedor.
## O projeto ainda está em desenvolvimento contínuo, novas animações, janelas, aplicativos e sistemas serão adicionados.

## Como rodar o projeto
Você pode utilizar tanto npm quanto yarn.

```bash
npm install
npm run dev
```

```bash
yarn
yarn dev
```

O projeto abrirá em http://localhost:5173 (ou outra porta gerada pelo Vite).
